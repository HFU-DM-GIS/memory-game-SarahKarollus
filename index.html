Memory Game

Anleitung

Das Ziel des Spiels ist es, Paare von Karten aufzudecken, die das gleiche Bild zeigen.

Das Spiel besteht aus 6, 8 oder 12 Kartenpaaren, die verdeckt auf dem Spielfeld liegen.

Um ein Thema zu wählen, einfach den Begriff in die suchzeile eingeben.

Du deckt zwei Karten auf, indem du auf sie klickt. Wenn die beiden Karten das gleiche Bild zeigen, bleiben sie aufgedeckt. Wenn die beiden Karten unterschiedliche Bilder haben, werden sie wieder verdeckt.

Das Spiel Endet, wenn du alle Paare gefunden wurden.

Viel Spaß beim Spielen!
