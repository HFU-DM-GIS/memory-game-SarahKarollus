Memory Game

Anleitung

Das Ziel des Spiels ist es, Paare von Karten aufzudecken, die das gleiche Bild zeigen.

Die Karten werden zu Beginn gemischt und verdeckt auf dem Spielfeld platziert. Der Spieler deckt eine Karte auf, indem er darauf klickt. Danach deckt er eine zweite Karte auf.

Wenn die beiden aufgedeckten Karten das gleiche Bild haben, bleiben sie aufgedeckt und der Spieler erhält einen Punkt. Wenn die beiden aufgedeckten Karten unterschiedliche Bilder haben, werden sie wieder verdeckt.

Das Spiel endet, wenn alle Paare gefunden wurden.
Viel Spaß beim Spielen!
